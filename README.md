Project Overview
This project analyzes Diwali season sales data to identify top-performing product categories, high-spending customer segments, and overall sales trends. The goal is to help businesses understand festive purchase behavior and improve marketing and inventory decisions.

Dataset
The dataset contains customer purchase records during the Diwali sales period.
It includes information such as:

* Customer demographics
* Product categories
* Order values
* Purchase trends

Dataset used: Diwali Sales Data.csv

Project Workflow

1. Loaded the dataset into Python using Pandas
2. Performed data cleaning by removing null values and correcting data types
3. Conducted exploratory data analysis (EDA)
4. Analyzed:

   * Sales by gender
   * Sales by age group
   * Top product categories
   * High-spending customer segments
5. Created visualizations to show key trends
6. Derived business insights from the analysis

Key Insights

* Identified top-performing product categories during Diwali
* Found high-value customer segments based on age and spending
* Observed purchase behavior trends useful for festive marketing strategies
